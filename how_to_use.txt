Move the files you want to upload into "files/" directory, then execute the script:

Ways of executing:

$ python3 gofile_uploader.py

$ python3 gofile_uploader.py tor

If you use tor parameter, then all requests will be done with tor, so make sure
tor is running on 127.0.0.1:9050

If you don't use tor parameter, then all requests will be done with your ACTUAL IP, so USE A VPN.
